# Keyword Blueprint

## Header File
- No Header File
- No main function
- No return 0

---

## Body

### 1. Comments

---

### 2. Built-in Functions

- bhan("Hello World"); → Print output  
- sun(variable); → Take user input  
- vayo → break keyword  
- jaujau / chalchal → continue keyword  
- firta → return keyword  
- gar → do keyword  

---

### 3. Data Types
Supported types:
- int
- float
- string
- bool
- array

Features:
- Loose-typed language

Keywords:
- manaum, yoho → declaration
- setgar → constant declaration

Example:
```plaintext
setgar a = 6;
```

---

### 4. Operators

- Arithmetic: + - * / % ++ --
- Assignment: = += -= *= /= %=
- Comparison: == != > < >= <=
- Logical: && || !
- Bitwise: & | ^ ~ << >>
- Member Access: . ->
- Others: :: * & ? : sizeof alignof new delete typeid
